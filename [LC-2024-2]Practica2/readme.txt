DESCRIPCIÓN DE IMPLEMENTACIÓN
--------------------------------------------------------------------------------
-- vars: Retorna una lista de todas las variables proposicionales presentes 
en una fórmula.
-- interp: Evalúa una fórmula proposicional dada un estado, es decir, asigna 
valores de verdad a las variables y calcula el valor de la fórmula resultante.
-- elimEquiv: Elimina las equivalencias lógicas de una fórmula.
-- elimImpl: Elimina las implicaciones lógicas de una fórmula.
-- posiblesInterp: Genera todas las posibles interpretaciones (estados) que 
una fórmula puede tener.
-- esModelo: Determina si un estado es un modelo de una fórmula proposicional.
-- todosModelos: Retorna una lista de todos los modelos (estados que hacen 
verdadera la fórmula) de una fórmula.
-- esSatisfacible: Determina si una fórmula es satisfacible, es decir, si 
tiene al menos un modelo.
-- esInsatisfacible: Determina si una fórmula es insatisfacible, es decir, 
si no tiene modelos.
-- esTautologia: Determina si una fórmula es una tautología, es decir, si su 
negación es insatisfacible.
-- esContradiccion: Determina si una fórmula es una contradicción, es decir, 
si su negación es una tautología.
